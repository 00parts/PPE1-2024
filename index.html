Présentation rapide du mini-projet encadré

Le but de ce mini-projet est de nous préparer à notre projet encadré final. Nous avons écrit un script qui prend en entrée un fichier texte contenant des URLs. Ce script ensuite va, à l'aide de plusieurs commandes telles que curl, lynx et wc, récupérer le code http de la requête, l'encodage de la page et enfin le nombre de mots dans la page. En sortie, nous avons un tableau qui regroupe ces résultats ligne par ligne pour chaque URL avec une numérotation pour chaque ligne.
